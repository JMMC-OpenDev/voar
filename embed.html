Embed samp applications in your web page

As first service we provide JavaScript API (V0.0.1) to display on your webpage a list of applications for a given list of SAMP MTypes. This API consists in a very basic jquery plugin. Please look at the sample snippet code provided just below to build list through following steps:

  • include JavaScript
  • declare your html list markup (<ul id="yourid"/> or <ol id="yourid"/>)
  • request filling
You can also look at our simple demo web page
